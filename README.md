# android-kotlin-lab

Kotlin 기반 Android 학습/예제 모음입니다.  
각 폴더는 **독립적인 예제 Android 프로젝트(또는 예제 코드)** 로 구성되어 있습니다.

## Projects

### DataStore
- `datastore/DataStroeRealEx` : DataStore 실습
- `datastore/StudyDataStore` : DataStore 학습

### Room
- `room/RoomWithFlow` : Room + Flow 학습

### Network
- `network/NetworkEx` : 네트워크 예제

### Architecture
- `architecture/CleanArchitectureEx` : Clean Architecture 예제
- `architecture/StudyMVI` : MVI 학습 예제

### RecyclerView
- `recyclerview/ListAdapter` : ListAdapter 예제
- `recyclerview/RecyclerviewEx` : RecyclerView 학습 예제

### ViewPager
- `viewpager/Vp2WithTab` : ViewPager2 + Tab 예제

### Testing
- `testing/Study` : 테스트 코드 학습 (TestCode)
  
## How to run
Android Studio에서 원하는 프로젝트 폴더를 열어 실행합니다.
(예: `room/RoomWithFlow`)
